# SliceAI UI 彻底美化 · 实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use `superpowers:subagent-driven-development` (recommended) or `superpowers:executing-plans` to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将 SliceAI 所有 UI 层迁移到新设计语言（Apple 原生 + 品牌紫 + 明暗主题），引入独立 `DesignSystem` target 集中 tokens / components，重构 5 大模块（悬浮工具栏 / 结果面板 / 命令面板 / 设置窗口 / Onboarding）。

**Architecture:** 新增 `DesignSystem` 作为 Package 的第 8 个 library target（零业务依赖、仅 SwiftUI + Foundation），承载 Colors / Typography / Spacing / Radius / Shadow / Animation / Material tokens + 通用 Components + Modifiers + ThemeManager。三个现有 UI target（Windowing / SettingsUI / Permissions）新增对 DesignSystem 的依赖；SliceCore 保持零 UI 依赖。主题数据通过 `Configuration.appearance` 字段持久化，`ThemeManager`（@Observable）作为环境注入驱动 UI。

**Tech Stack:** Swift 6.0 严格并发 / SwiftUI (macOS 14+) / Swift Observation (`@Observable`) / NSVisualEffectView / Xcode Asset Catalog / SwiftLint (strict) / SwiftPM local package

**Reference Spec:** `docs/superpowers/specs/2026-04-21-ui-polish-design.md`（所有视觉参数、颜色值、交互细节以 spec 为准，本 plan 只列关键片段，过长的数据表请查 spec）

**Working Directory:** 所有 Swift 命令在 `SliceAIKit/` 子目录执行；git / 文档 / asset 命令在仓库根目录执行。

## 全局约定

- **命名**：所有 DesignSystem 公开类型以 `Slice` 前缀（SliceColor / SliceFont / SliceSpacing / SliceRadius / SliceShadow / SliceAnimation / SliceTransition / SliceMaterial）；Modifiers 以 View extension 形式暴露；Components 直接命名（IconButton / PillButton 等）
- **注释**：每个 public 类型 / 函数必须带 `///` 三斜线文档注释（已由 .swift-format 配置强制）；关键中文注释保留，方便代码审查
- **日志**：按 CLAUDE.md 规则，关键路径加 `print`（调试用，后续统一切到 os.Logger 是未来工作）
- **SwiftLint**：每次改动后必须 `swiftlint lint --strict` 零警告
- **测试**：可单测的逻辑（ThemeManager / Configuration / 纯算法）走 TDD；纯视觉改动写"视觉验收步骤"而非测试
- **Commit 规范**：每个 task 一个 commit；信息遵循现有仓库风格（中文 or 英文均可，保持一致）；commit message 带 `ui:` / `feat:` / `refactor:` 前缀
- **回滚便利**：每个 task 自成可回滚的单元（build + test 绿），以便任何一步出问题可 `git revert` 单个 commit

- [ ] **Step 2: 编写 SliceColor.swift**

```swift
// SliceAIKit/Sources/DesignSystem/Colors/SliceColor.swift
import SwiftUI

/// 语义化色板，统一从 DesignSystem 的 Assets.xcassets 加载
///
/// 所有 Color 走 Xcode Asset Catalog 的 "Any / Dark" 变体机制，SwiftUI 自动
/// 根据当前 `colorScheme` 切换。禁止在业务代码中硬编码 `Color(red:green:blue:)`，
/// 应一律通过 `SliceColor.xxx` 访问以便主题切换生效。
///
/// 使用示例：
/// ```swift
/// Text("Hello")
///     .foregroundColor(SliceColor.textPrimary)
///     .background(SliceColor.surface)
/// ```
public enum SliceColor {
    // MARK: 背景层
    /// 窗口底色（最底层）
    public static let background       = Color("background", bundle: .module)
    /// 卡片 / 面板底色
    public static let surface          = Color("surface", bundle: .module)
    /// 悬浮层 / HUD 底色（带透明度，通常叠加毛玻璃）
    public static let surfaceElevated  = Color("surfaceElevated", bundle: .module)

    // MARK: 分隔与边框
    /// 0.5pt 细分隔线
    public static let divider          = Color("divider", bundle: .module)
    /// 外框描边
    public static let border           = Color("border", bundle: .module)

    // MARK: 文本层级
    /// 主文本
    public static let textPrimary      = Color("textPrimary", bundle: .module)
    /// 次文本
    public static let textSecondary    = Color("textSecondary", bundle: .module)
    /// 辅助 / 三级文本
    public static let textTertiary     = Color("textTertiary", bundle: .module)
    /// 禁用文本
    public static let textDisabled     = Color("textDisabled", bundle: .module)

    // MARK: 品牌强调
    /// 主强调色（紫）
    public static let accent           = Color("accent", bundle: .module)
    /// 浅填充（hover / chip 底 / selected row 底）
    public static let accentFillLight  = Color("accentFillLight", bundle: .module)
    /// 深填充（active / pressed / icon 底）
    public static let accentFillStrong = Color("accentFillStrong", bundle: .module)
    /// 紫色文字（link / active tab / 选中态 label）
    public static let accentText       = Color("accentText", bundle: .module)

    // MARK: 状态色
    /// 错误
    public static let error            = Color("error", bundle: .module)
    /// 错误填充（error block 底）
    public static let errorFill        = Color("errorFill", bundle: .module)
    /// 错误边框
    public static let errorBorder      = Color("errorBorder", bundle: .module)
    /// 成功
    public static let success          = Color("success", bundle: .module)
    /// 警告
    public static let warning          = Color("warning", bundle: .module)
    /// 警告填充
    public static let warningFill      = Color("warningFill", bundle: .module)

    // MARK: 交互反馈
    /// 通用 hover 浅色
    public static let hoverFill        = Color("hoverFill", bundle: .module)
    /// 通用按压深色
    public static let pressedFill      = Color("pressedFill", bundle: .module)
}
```

// SliceAIKit/Sources/DesignSystem/Materials/VisualEffectView.swift
import AppKit
import SwiftUI

/// SwiftUI 对 `NSVisualEffectView` 的薄封装
///
/// SwiftUI macOS 原生虽然提供 `.regularMaterial` 等 `Material` 语义，但 `NSVisualEffectView`
/// 可以精细控制 `material`（hudWindow / sidebar / popover / windowBackground）和
/// `blendingMode`（behindWindow / withinWindow），对 HUD 窗口观感更可控。
///
/// 使用方式：
/// ```swift
/// SomeView()
///     .background(VisualEffectView(material: .hudWindow))
/// ```
/// 或通过便利 modifier `.glassBackground(.hud)`（见 Modifiers/GlassBackground.swift）。
public struct VisualEffectView: NSViewRepresentable {
    /// 毛玻璃材质类型
    public let material: NSVisualEffectView.Material
    /// 混合模式：behindWindow 透出桌面；withinWindow 透出同窗口底层
    public let blendingMode: NSVisualEffectView.BlendingMode
    /// 活动状态：固定 `.active` 保证毛玻璃不因窗口失焦变暗
    public let state: NSVisualEffectView.State

    /// 构造
    /// - Parameters:
    ///   - material: 材质类型，默认 hudWindow（最适合浮层）
    ///   - blendingMode: 混合模式，默认 behindWindow
    ///   - state: 活动状态，默认 active（避免失焦变浊）
    public init(
        material: NSVisualEffectView.Material = .hudWindow,
        blendingMode: NSVisualEffectView.BlendingMode = .behindWindow,
        state: NSVisualEffectView.State = .active
    ) {
        self.material = material
        self.blendingMode = blendingMode
        self.state = state
    }

    /// 创建底层 NSVisualEffectView
    public func makeNSView(context: Context) -> NSVisualEffectView {
        let view = NSVisualEffectView()
        view.material = material
        view.blendingMode = blendingMode
        view.state = state
        return view
    }

    /// 响应 SwiftUI 状态变更（material / blending 切换时同步到 AppKit 层）
    public func updateNSView(_ nsView: NSVisualEffectView, context: Context) {
        nsView.material = material
        nsView.blendingMode = blendingMode
        nsView.state = state
    }
}

// SliceAIKit/Sources/DesignSystem/Theme/ThemeManager.swift
import AppKit
import Observation
import SwiftUI

/// 主题管理器（Swift Observation 驱动）
///
/// 职责：
/// - 持有当前 `AppearanceMode`（auto / light / dark）
/// - 解析出 SwiftUI 层的 `ColorScheme` 与 AppKit 层的 `NSAppearance`
/// - 通过 `onModeChange` 回调通知外部（通常为 ConfigurationStore）持久化
///
/// 线程模型：`@MainActor` 限定。由 AppContainer 构造后注入到根视图的 environment。
/// 使用方式：
/// ```swift
/// // AppContainer 构造
/// let themeManager = ThemeManager(initialMode: config.appearance)
/// themeManager.onModeChange = { configStore.updateAppearance($0) }
///
/// // 根视图注入
/// ContentView().environment(themeManager)
///
/// // 消费
/// @Environment(ThemeManager.self) private var theme
/// ```
@MainActor
@Observable
public final class ThemeManager {

    /// 当前主题模式（写入时触发 @Observable 重绘）
    public var mode: AppearanceMode {
        didSet {
            guard oldValue != mode else { return }
            onModeChange?(mode)
        }
    }

    /// 当 mode 改变时的回调（通常让 ConfigurationStore 持久化）
    public var onModeChange: (@MainActor (AppearanceMode) -> Void)?

    /// 构造
    /// - Parameter initialMode: 初始模式，通常从 Configuration.appearance 加载
    public init(initialMode: AppearanceMode = .auto) {
        self.mode = initialMode
    }

    /// 显式切换模式；等价于 `mode = newValue`，但语义更清晰
    /// - Parameter newMode: 新模式
    public func setMode(_ newMode: AppearanceMode) {
        self.mode = newMode
    }

    /// 解析出 SwiftUI `ColorScheme`。
    /// `.auto` 时返回 `.light`（SwiftUI 默认值），由上层决定是否覆盖；
    /// 实际渲染在 `.preferredColorScheme(nil)` 时会跟随系统。
    public var resolvedColorScheme: ColorScheme {
        switch mode {
        case .light: return .light
        case .dark:  return .dark
        case .auto:  return .light
        }
    }

    /// 显式 NSAppearance（绑定到 NSWindow 的 appearance 属性）
    /// `.auto` 返回 nil，让 window 自动跟随系统（`NSWindow.appearance = nil`）
    public var nsAppearance: NSAppearance? {
        switch mode {
        case .light: return NSAppearance(named: .aqua)
        case .dark:  return NSAppearance(named: .darkAqua)
        case .auto:  return nil
        }
    }

    /// 将当前主题应用到 NSWindow（auto 时清空 appearance 让其跟随系统）
    /// - Parameter window: 目标 NSWindow
    public func apply(to window: NSWindow) {
        window.appearance = nsAppearance
    }
}

```swift
public struct Configuration: Sendable, Codable, Equatable {
    public let schemaVersion: Int
    public var providers: [Provider]
    public var tools: [Tool]
    public var hotkeys: HotkeyBindings
    public var triggers: TriggerSettings
    public var telemetry: TelemetrySettings
    public var appBlocklist: [String]
    /// 应用主题模式；缺失（旧 config）时默认 .auto
    public var appearance: AppearanceMode

    public init(schemaVersion: Int, providers: [Provider], tools: [Tool],
                hotkeys: HotkeyBindings, triggers: TriggerSettings,
                telemetry: TelemetrySettings, appBlocklist: [String],
                appearance: AppearanceMode = .auto) {
        self.schemaVersion = schemaVersion
        self.providers = providers
        self.tools = tools
        self.hotkeys = hotkeys
        self.triggers = triggers
        self.telemetry = telemetry
        self.appBlocklist = appBlocklist
        self.appearance = appearance
    }

    public static let currentSchemaVersion = 1

    // MARK: - 向后兼容解码

    /// 旧版本 JSON 可能缺少 appearance 字段；用自定义 init(from:) 回落到 .auto
    public init(from decoder: Decoder) throws {
        let c = try decoder.container(keyedBy: CodingKeys.self)
        self.schemaVersion = try c.decode(Int.self, forKey: .schemaVersion)
        self.providers = try c.decode([Provider].self, forKey: .providers)
        self.tools = try c.decode([Tool].self, forKey: .tools)
        self.hotkeys = try c.decode(HotkeyBindings.self, forKey: .hotkeys)
        self.triggers = try c.decode(TriggerSettings.self, forKey: .triggers)
        self.telemetry = try c.decode(TelemetrySettings.self, forKey: .telemetry)
        self.appBlocklist = try c.decode([String].self, forKey: .appBlocklist)
        self.appearance = try c.decodeIfPresent(AppearanceMode.self, forKey: .appearance) ?? .auto
    }

    private enum CodingKeys: String, CodingKey {
        case schemaVersion, providers, tools, hotkeys, triggers, telemetry, appBlocklist, appearance
    }
}
```

    startTrackingTheme()

private func startTrackingTheme() {
    withObservationTracking {
        _ = container.themeManager.mode
    } onChange: { [weak self] in
        Task { @MainActor in
            self?.applyAppearanceToAllWindows()
            self?.startTrackingTheme()  // 重新订阅
        }
    }
}

private func applyAppearanceToAllWindows() {
    let appearance = container.themeManager.nsAppearance
    for window in NSApp.windows {
        window.appearance = appearance
    }
}

// SliceAIKit/Sources/DesignSystem/Modifiers/HoverHighlight.swift
import SwiftUI

/// Hover 时叠加浅填充背景的 modifier
///
/// 用于图标按钮、列表项等希望鼠标悬停时有轻微视觉反馈的场景。默认使用 `SliceColor.hoverFill`；
/// 传入自定义 fill 以实现强调色（如 accent 悬停）。
public struct HoverHighlight: ViewModifier {
    let fill: Color
    let cornerRadius: CGFloat
    @State private var isHovered = false

    public func body(content: Content) -> some View {
        content
            .background(
                RoundedRectangle(cornerRadius: cornerRadius)
                    .fill(isHovered ? fill : .clear)
            )
            .onHover { hovering in
                withAnimation(SliceAnimation.quick) { isHovered = hovering }
            }
    }
}

public extension View {
    /// 附加 hover 高亮
    /// - Parameters:
    ///   - fill: 悬停填充色，默认 `SliceColor.hoverFill`
    ///   - cornerRadius: 圆角，默认 `SliceRadius.button`
    func hoverHighlight(
        fill: Color = SliceColor.hoverFill,
        cornerRadius: CGFloat = SliceRadius.button
    ) -> some View {
        modifier(HoverHighlight(fill: fill, cornerRadius: cornerRadius))
    }
}

/// 按下时缩小到 0.94 倍的 modifier（物理按压感）
///
/// 适用于 Button label。直接套在 Button label 外即可：
/// ```swift
/// Button(...) { ... }.buttonStyle(PressScaleButtonStyle())
/// ```
/// 或通过 `.pressScale()` 便利扩展。
public struct PressScaleButtonStyle: ButtonStyle {

            .scaleEffect(configuration.isPressed ? scale : 1.0)

    }
}

public extension View {
    /// 附加按压缩放的 Button style
    func pressScale(_ scale: CGFloat = 0.94) -> some View {
        buttonStyle(PressScaleButtonStyle(scale: scale))
    }

import SwiftUI

/// 悬浮工具栏左侧的拖拽把手
///
/// 视觉：2×3 共 6 个小圆点，直径 1.8pt、间距 4pt，hover 时底色变浅 + 游标切 openHand。
/// 交互：点击拖动时由外层调用 `NSWindow.performDrag(with:)` 实现整窗移动；本组件仅负责视觉与游标。
///
/// 使用方式：
/// ```swift
/// DragHandle()
///     .onDragGesture { /* trigger NSWindow.performDrag */ }
/// ```
public struct DragHandle: View {
    /// 是否正在被拖拽（由外层控制，影响点的配色）
    public let isDragging: Bool
    /// Hover 状态（内部自管）
    @State private var isHovered = false

    public init(isDragging: Bool = false) {
        self.isDragging = isDragging
    }

    public var body: some View {
        HStack(spacing: 0) {
            Canvas { ctx, size in
                let dot = CGSize(width: 1.8, height: 1.8)
                let step: CGFloat = 4
                let originX = (size.width - dot.width - step) / 2
                let originY = (size.height - dot.height * 3 - step * 2) / 2
                for col in 0..<2 {
                    for row in 0..<3 {
                        let rect = CGRect(
                            x: originX + CGFloat(col) * step,
                            y: originY + CGFloat(row) * step,
                            width: dot.width, height: dot.height
                        )
                        ctx.fill(Path(ellipseIn: rect), with: .color(dotColor))
                    }
                }
            }
            .frame(width: 14, height: 28)
            .background(
                RoundedRectangle(cornerRadius: SliceRadius.tight)
                    .fill(backgroundFill)
            )
            .onHover { hovering in
                isHovered = hovering
                if hovering { NSCursor.openHand.push() } else { NSCursor.pop() }
            }
            // 右侧分隔竖线，16pt 高 / 1pt 宽
            Rectangle()
                .fill(SliceColor.divider)
                .frame(width: 1, height: 16)
                .padding(.horizontal, 3)
        }
    }

    private var dotColor: Color {
        isDragging ? SliceColor.accent : SliceColor.textTertiary
    }

    private var backgroundFill: Color {
        if isDragging { return SliceColor.accentFillLight }
        if isHovered  { return SliceColor.hoverFill }
        return .clear
    }
}

/// 三个脉动小点 + "正在思考…" 文案
///
/// 动画：每个点 1.4s 循环脉动，相邻点延迟 0.2s，合成 "波浪"。

    @State private var phase: Int = 0

    public init(label: String = "正在思考…") {
        self.label = label
    }

    public var body: some View {
        HStack(spacing: SliceSpacing.base) {
            HStack(spacing: 3) {
                ForEach(0..<3, id: \.self) { i in
                    Circle()
                        .fill(SliceColor.accent)
                        .frame(width: 5, height: 5)
                        .opacity(opacity(for: i))
                        .scaleEffect(scale(for: i))
                }
            }
            Text(label)
                .font(SliceFont.body)
                .foregroundColor(SliceColor.textSecondary)
        }
        .onAppear { startAnimation() }
    }

    private func opacity(for index: Int) -> Double {
        let active = (phase == index)
        return active ? 1.0 : 0.3
    }

    private func scale(for index: Int) -> Double {
        let active = (phase == index)
        return active ? 1.1 : 0.8
    }

    private func startAnimation() {
        // 每 470ms 切一次（3 点 × 470ms ≈ 1.4s 一个完整循环）
        Timer.scheduledTimer(withTimeInterval: 0.47, repeats: true) { _ in
            withAnimation(.easeInOut(duration: 0.47)) {
                phase = (phase + 1) % 3
            }
        }
    }
}

import SwiftUI

/// Onboarding 的三步进度指示器
public struct StepIndicator: View {
    public struct Step: Identifiable {
        public let id: Int
        public let label: String
        public init(id: Int, label: String) { self.id = id; self.label = label }
    }

    let steps: [Step]
    let currentIndex: Int  // 0-based

    public init(steps: [Step], currentIndex: Int) {
        self.steps = steps
        self.currentIndex = currentIndex
    }

    public var body: some View {
        HStack(spacing: SliceSpacing.lg) {
            ForEach(Array(steps.enumerated()), id: \.element.id) { idx, step in
                nodeView(idx: idx, step: step)
                if idx < steps.count - 1 {
                    connector(active: idx < currentIndex)
                }
            }
        }
    }

    @ViewBuilder
    private func nodeView(idx: Int, step: Step) -> some View {
        let state: NodeState = idx < currentIndex ? .done :
                               idx == currentIndex ? .active : .pending
        HStack(spacing: SliceSpacing.md) {
            ZStack {
                Circle().fill(state.fill)
                if state == .done {
                    Image(systemName: "checkmark")
                        .font(.system(size: 11, weight: .bold))
                        .foregroundColor(SliceColor.accentText)
                } else {
                    Text("\(step.id)")
                        .font(SliceFont.captionEmphasis)
                        .foregroundColor(state.textColor)
                }
            }
            .frame(width: 20, height: 20)
            .overlay(
                Circle().stroke(state == .active ? SliceColor.accentFillLight : .clear, lineWidth: 4)
            )
            Text(step.label.uppercased())
                .font(SliceFont.overline)
                .kerning(SliceKerning.wide)
                .foregroundColor(state.labelColor)
        }
    }

    private func connector(active: Bool) -> some View {
        Rectangle()
            .fill(active ? SliceColor.accent.opacity(0.4) : SliceColor.divider)
            .frame(width: 38, height: 1.5)
    }

    private enum NodeState {
        case pending, active, done
        var fill: Color {
            switch self {
            case .pending: return SliceColor.hoverFill
            case .active:  return SliceColor.accent
            case .done:    return SliceColor.accentFillLight
            }
        }
        var textColor: Color {
            switch self {
            case .pending: return SliceColor.textTertiary
            case .active:  return .white
            case .done:    return SliceColor.accentText
            }
        }
        var labelColor: Color {
            switch self {
            case .active: return SliceColor.accentText
            default:      return SliceColor.textTertiary
            }

/// 划词后弹出的紧贴选区浮条（A 模式）· 美化版
@MainActor
public final class FloatingToolbarPanel {

    private var panel: NSPanel?
    private let positioner = ScreenAwarePositioner()
    private var autoDismissTask: Task<Void, Never>?
    private var outsideClickMonitor: Any?

    public init() {}

    public func show(tools: [Tool], anchor: CGPoint, onPick: @escaping (Tool) -> Void) {
        // 宽度 = 把手 (14 + 分隔 7) + 每个按钮 30pt + 间距 2pt + 左右 padding 4pt ×2
        let handleWidth: CGFloat = 14 + 7
        let buttonsWidth = CGFloat(tools.count) * 30 + CGFloat(max(0, tools.count - 1)) * 2
        let padding: CGFloat = 4
        let width = handleWidth + buttonsWidth + padding * 2
        let height: CGFloat = 30 + padding * 2
        let size = CGSize(width: max(width, 120), height: height)

        let screen = NSScreen.screens.first(where: {
            $0.visibleFrame.contains(anchor)
        })?.visibleFrame ?? NSScreen.main?.visibleFrame ?? .zero
        let origin = positioner.position(anchor: anchor, size: size, screen: screen, offset: 8)

        let panel = makePanel(size: size, origin: origin)

        let content = ToolbarContent(
            tools: tools,
            onPick: { [weak self] t in onPick(t); self?.dismiss() },
            pauseAutoDismiss: { [weak self] in self?.autoDismissTask?.cancel() },
            resumeAutoDismiss: { [weak self] in self?.scheduleAutoDismiss() },
            requestDrag: { [weak panel] event in
                panel?.performDrag(with: event)
            }
        )
        let hosting = NSHostingView(rootView: content)
        hosting.frame = NSRect(origin: .zero, size: size)
        panel.contentView = hosting
        panel.orderFrontRegardless()
        self.panel = panel

        scheduleAutoDismiss()
        installOutsideClickMonitor()
    }

    public func dismiss() {
        autoDismissTask?.cancel()
        removeOutsideClickMonitor()
        panel?.orderOut(nil)
        panel = nil
    }

    private func scheduleAutoDismiss() {
        autoDismissTask?.cancel()
        autoDismissTask = Task { @MainActor [weak self] in
            try? await Task.sleep(nanoseconds: 5_000_000_000)
            guard !Task.isCancelled else { return }
            self?.dismiss()
        }
    }

    private func installOutsideClickMonitor() {
        removeOutsideClickMonitor()
        outsideClickMonitor = NSEvent.addGlobalMonitorForEvents(
            matching: [.leftMouseDown, .rightMouseDown, .otherMouseDown]
        ) { [weak self] _ in
            Task { @MainActor in self?.dismiss() }
        }
    }

    private func removeOutsideClickMonitor() {
        if let monitor = outsideClickMonitor {
            NSEvent.removeMonitor(monitor)
            outsideClickMonitor = nil
        }
    }

    private func makePanel(size: CGSize, origin: CGPoint) -> NSPanel {
        let panel = NSPanel(
            contentRect: NSRect(origin: origin, size: size),
            styleMask: [.borderless, .nonactivatingPanel],
            backing: .buffered, defer: false
        )
        panel.isFloatingPanel = true
        panel.level = .statusBar
        panel.collectionBehavior = [.canJoinAllSpaces, .fullScreenAuxiliary]
        panel.hidesOnDeactivate = false
        panel.isMovable = true
        panel.isMovableByWindowBackground = false  // 改为仅把手区拖动
        panel.hasShadow = true
        panel.backgroundColor = .clear
        panel.isOpaque = false
        return panel
    }
}

/// 浮条 SwiftUI 视图
private struct ToolbarContent: View {
    let tools: [Tool]
    let onPick: (Tool) -> Void
    let pauseAutoDismiss: () -> Void
    let resumeAutoDismiss: () -> Void
    let requestDrag: (NSEvent) -> Void

    @State private var isDragging = false

    var body: some View {
        HStack(spacing: 2) {
            // 拖拽把手（hover 变 openHand，按下转发给 NSWindow.performDrag）
            DragHandle(isDragging: isDragging)
                .background(DragGestureHost { event, phase in
                    switch phase {
                    case .began:

private struct DragGestureHost: NSViewRepresentable {

    let onDrag: (NSEvent, Phase) -> Void

        let view = MouseCaptureView()
        view.onDrag = onDrag
        return view
    }
    func updateNSView(_ nsView: MouseCaptureView, context: Context) {
        nsView.onDrag = onDrag
    }

        var onDrag: ((NSEvent, Phase) -> Void)?
        override func mouseDown(with event: NSEvent) {
            onDrag?(event, .began)
            super.mouseDown(with: event)
            onDrag?(event, .ended)
        }
    }

**Files:**

因为 `AttributedString(markdown:)` 的 `.full` 模式需要完整文档，流式期间尾部可能是未闭合语法导致解析失败。解决方案：**尝试 `.full` 解析；失败则回落到 `.inlineOnlyPreservingWhitespace`；最终回落到纯文本**。

- [ ] **Step 1: 重写 StreamingMarkdownView**

```swift
// SliceAIKit/Sources/Windowing/StreamingMarkdownView.swift
import DesignSystem
import SwiftUI

/// 流式 Markdown 渲染视图
///
/// 解析策略（三级回退）：
/// 1. `.full` 完整解析：支持代码块 / 列表 / 标题 / 引用
/// 2. `.inlineOnlyPreservingWhitespace`：保留换行，只做行内 markdown
/// 3. 纯文本：全部当成原文
///
/// 为了处理 ```代码块``` 与 `> quote` 等块级元素，本实现采用**分段解析**：
/// 按双换行切段，每段独立尝试解析，这样不会因局部未闭合影响整体。
public struct StreamingMarkdownView: View {
    public let text: String
    public let isStreaming: Bool

    public init(text: String, isStreaming: Bool) {
        self.text = text
        self.isStreaming = isStreaming
    }

    public var body: some View {
        ScrollView {
            VStack(alignment: .leading, spacing: SliceSpacing.lg) {
                ForEach(Array(blocks.enumerated()), id: \.offset) { _, block in
                    renderBlock(block)
                }
                if isStreaming {
                    BlinkingCursor()
                }
            }
            .frame(maxWidth: .infinity, alignment: .leading)
            .padding(.horizontal, SliceSpacing.xxl)
            .padding(.vertical, SliceSpacing.xl)
        }
    }

    // MARK: 分段

    /// 按空行切分成 block；每个 block 是 (kind, payload)
    private var blocks: [Block] { parseBlocks(text) }

    private enum Block {
        case paragraph(String)
        case heading(level: Int, text: String)
        case codeBlock(String)
        case quote(String)
        case list(items: [String])
    }

    private func parseBlocks(_ raw: String) -> [Block] {
        var result: [Block] = []
        // 代码块用 fence 匹配（```...```）
        var lines = raw.components(separatedBy: "\n")
        var i = 0
        while i < lines.count {
            let line = lines[i]
            // 代码围栏
            if line.trimmingCharacters(in: .whitespaces).hasPrefix("```") {
                var code: [String] = []
                i += 1
                while i < lines.count, !lines[i].trimmingCharacters(in: .whitespaces).hasPrefix("```") {
                    code.append(lines[i])
                    i += 1
                }
                // 跳过结尾的 ```
                if i < lines.count { i += 1 }
                result.append(.codeBlock(code.joined(separator: "\n")))
                continue
            }
            // 标题
            if let h = parseHeading(line) {
                result.append(h); i += 1; continue
            }
            // 引用
            if line.hasPrefix("> ") {
                var q = [String(line.dropFirst(2))]
                i += 1
                while i < lines.count, lines[i].hasPrefix("> ") {
                    q.append(String(lines[i].dropFirst(2)))
                    i += 1
                }
                result.append(.quote(q.joined(separator: "\n")))
                continue
            }
            // 列表
            if line.hasPrefix("- ") || line.hasPrefix("* ") {
                var items = [String(line.dropFirst(2))]
                i += 1
                while i < lines.count, lines[i].hasPrefix("- ") || lines[i].hasPrefix("* ") {
                    items.append(String(lines[i].dropFirst(2)))
                    i += 1
                }
                result.append(.list(items: items))
                continue
            }
            // 普通段落：往下吃到空行
            var para = [line]
            i += 1
            while i < lines.count, !lines[i].isEmpty, !specialPrefix(lines[i]) {
                para.append(lines[i])
                i += 1
            }
            let joined = para.joined(separator: "\n").trimmingCharacters(in: .whitespaces)
            if !joined.isEmpty { result.append(.paragraph(joined)) }
            // 跳过空行
            while i < lines.count, lines[i].isEmpty { i += 1 }
        }
        return result
    }

    private func specialPrefix(_ line: String) -> Bool {
        line.hasPrefix("#") || line.hasPrefix("> ") ||
        line.hasPrefix("- ") || line.hasPrefix("* ") ||
        line.trimmingCharacters(in: .whitespaces).hasPrefix("```")
    }

    private func parseHeading(_ line: String) -> Block? {
        var level = 0
        var s = line
        while s.hasPrefix("#") { level += 1; s.removeFirst() }
        guard level >= 1, level <= 4, s.hasPrefix(" ") else { return nil }
        return .heading(level: level, text: String(s.dropFirst()))
    }

    // MARK: 渲染

    @ViewBuilder
    private func renderBlock(_ block: Block) -> some View {
        switch block {
        case .paragraph(let s):
            inlineText(s)
                .font(SliceFont.body)
                .kerning(SliceKerning.normal)
                .lineSpacing(SliceLineSpacing.body)
                .foregroundColor(SliceColor.textPrimary)
        case .heading(let level, let s):
            Text(s)
                .font(.system(size: headingSize(level), weight: .bold))
                .kerning(SliceKerning.snug)
                .foregroundColor(SliceColor.textPrimary)
                .padding(.top, level <= 2 ? SliceSpacing.base : SliceSpacing.sm)
        case .codeBlock(let s):
            ScrollView(.horizontal, showsIndicators: false) {
                Text(s)
                    .font(SliceFont.mono)
                    .foregroundColor(SliceColor.textPrimary)
                    .padding(.horizontal, SliceSpacing.xl)
                    .padding(.vertical, SliceSpacing.lg)
            }
            .frame(maxWidth: .infinity, alignment: .leading)
            .background(
                RoundedRectangle(cornerRadius: SliceRadius.control).fill(SliceColor.hoverFill)
            )
        case .quote(let s):
            HStack(alignment: .top, spacing: SliceSpacing.lg) {
                Rectangle().fill(SliceColor.accent).frame(width: 2)
                inlineText(s)
                    .font(SliceFont.body.italic())
                    .foregroundColor(SliceColor.textSecondary)
                    .opacity(0.85)
            }
            .padding(.leading, 0)
        case .list(let items):
            VStack(alignment: .leading, spacing: SliceSpacing.xs) {
                ForEach(Array(items.enumerated()), id: \.offset) { _, item in
                    HStack(alignment: .top, spacing: SliceSpacing.base) {
                        Text("•").foregroundColor(SliceColor.textTertiary)
                        inlineText(item)
                            .font(SliceFont.body)
                            .foregroundColor(SliceColor.textPrimary)
                    }
                }
            }
            .padding(.leading, SliceSpacing.base)
        }
    }

    private func headingSize(_ level: Int) -> CGFloat {
        switch level {
        case 1: return 17
        case 2: return 16
        case 3: return 15
        default: return 14
        }
    }

    /// 段落内 markdown：尝试 .inlineOnly 解析，失败回落纯文本
    @ViewBuilder
    private func inlineText(_ s: String) -> some View {
        if let attr = try? AttributedString(
            markdown: s,
            options: AttributedString.MarkdownParsingOptions(
                interpretedSyntax: .inlineOnlyPreservingWhitespace
            )
        ) {
            Text(attr).textSelection(.enabled)
        } else {
            Text(s).textSelection(.enabled)
        }
    }
}

/// 闪烁光标：6×13pt 紫色方块，1Hz 闪烁

    @State private var visible = true
    @State private var blinker: Task<Void, Never>?

                }

    }
}

/// ⌥Space 调出的命令面板 · 美化版
@MainActor
public final class CommandPalettePanel {
    private var panel: NSPanel?
    public init() {}

    public func show(tools: [Tool], preview: String?, onPick: @escaping (Tool) -> Void) {
        let size = CGSize(width: 560, height: 420)
        let screen = NSScreen.main?.visibleFrame ?? .zero
        // 垂直居中偏上 30%（Raycast 经典位置）
        let origin = CGPoint(
            x: screen.midX - size.width / 2,
            y: screen.minY + screen.height * 0.55
        )
        let panel = makePanel(size: size, origin: origin)
        let hosting = NSHostingView(rootView: PaletteContent(
            tools: tools, preview: preview ?? "",
            onPick: { [weak self] t in onPick(t); self?.dismiss() },
            onCancel: { [weak self] in self?.dismiss() }
        ))
        hosting.frame = NSRect(origin: .zero, size: size)
        hosting.autoresizingMask = [.width, .height]
        panel.contentView = hosting
        panel.makeKeyAndOrderFront(nil)
        NSApp.activate(ignoringOtherApps: true)
        self.panel = panel
    }

    public func dismiss() { panel?.orderOut(nil); panel = nil }

    private func makePanel(size: CGSize, origin: CGPoint) -> NSPanel {
        let panel = NSPanel(
            contentRect: NSRect(origin: origin, size: size),
            styleMask: [.titled, .fullSizeContentView, .utilityWindow],
            backing: .buffered, defer: false
        )
        panel.titleVisibility = .hidden
        panel.titlebarAppearsTransparent = true
        panel.isMovableByWindowBackground = true
        panel.level = .floating
        panel.collectionBehavior = [.canJoinAllSpaces]
        panel.backgroundColor = .clear
        panel.isOpaque = false
        panel.hasShadow = true
        return panel
    }
}

            if !preview.isEmpty {
                HStack(spacing: SliceSpacing.md) {
                    Text("选中文本")
                        .font(SliceFont.overline)
                        .kerning(SliceKerning.wide)
                        .foregroundColor(SliceColor.textTertiary)

                        .font(SliceFont.caption)

                .padding(.top, SliceSpacing.base)
                .padding(.bottom, SliceSpacing.md)

                    .font(.system(size: 15))

        case .accessibility: step = .apiKey
        case .apiKey:        onFinish(apiKey)
        }
    }

import AppKit

@MainActor
final class MenuBarController {
    weak var delegate: AppDelegate?
    private let statusItem: NSStatusItem
    private let themeManager: ThemeManager
    private weak var container: AppContainer?

    init(container: AppContainer, delegate: AppDelegate) {
        self.delegate = delegate
        self.themeManager = container.themeManager
        self.container = container
        statusItem = NSStatusBar.system.statusItem(withLength: NSStatusItem.squareLength)
        statusItem.button?.image = NSImage(
            systemSymbolName: "scissors",
            accessibilityDescription: "SliceAI"
        )
        statusItem.menu = buildMenu()
        // 订阅配置变化 → 更新红点（未配置 provider 时提示）
        refreshConfigStateIndicator()
    }

    /// 依据当前配置刷新 menu bar 图标（无 provider 时加小红点）
    func refreshConfigStateIndicator() {
        guard let providers = container?.configStore.configuration.providers else { return }
        let hasAny = !providers.isEmpty
        statusItem.button?.image = hasAny ? baseIcon() : badgedIcon()
    }

    private func baseIcon() -> NSImage? {
        NSImage(systemSymbolName: "scissors", accessibilityDescription: "SliceAI")
    }

    private func badgedIcon() -> NSImage? {
        guard let base = baseIcon() else { return nil }
        let size = base.size
        let img = NSImage(size: size, flipped: false) { rect in
            base.draw(in: rect)
            // 右上角小红点（紫色用于品牌一致性）
            let dotSize: CGFloat = 6
            let dotRect = NSRect(
                x: rect.maxX - dotSize, y: rect.maxY - dotSize,
                width: dotSize, height: dotSize
            )
            NSColor(red: 124/255, green: 58/255, blue: 237/255, alpha: 1).setFill()
            NSBezierPath(ovalIn: dotRect).fill()
            return true
        }
        img.isTemplate = false  // 带色，不走系统 template tint
        return img
    }

    private func buildMenu() -> NSMenu {
        let menu = NSMenu()
        menu.addItem(NSMenuItem(title: "SliceAI", action: nil, keyEquivalent: ""))
        menu.addItem(.separator())
        menu.addItem(
            NSMenuItem(title: "Settings…",
                       action: #selector(openSettings),
                       keyEquivalent: ",").withTarget(self)
        )
        // 外观子菜单
        let appearanceItem = NSMenuItem(title: "外观", action: nil, keyEquivalent: "")
        appearanceItem.submenu = buildAppearanceMenu()
        menu.addItem(appearanceItem)
        menu.addItem(.separator())
        menu.addItem(
            NSMenuItem(title: "Quit",
                       action: #selector(NSApplication.terminate(_:)),
                       keyEquivalent: "q")
        )
        return menu
    }

    private func buildAppearanceMenu() -> NSMenu {
        let sub = NSMenu()
        for mode in AppearanceMode.allCases {
            let item = NSMenuItem(
                title: mode.displayName,
                action: #selector(setAppearance(_:)),
                keyEquivalent: ""
            )
            item.target = self
            item.representedObject = mode.rawValue
            item.state = themeManager.mode == mode ? .on : .off
            sub.addItem(item)
        }
        return sub
    }

    @objc private func setAppearance(_ sender: NSMenuItem) {
        guard let raw = sender.representedObject as? String,
              let mode = AppearanceMode(rawValue: raw) else { return }
        themeManager.setMode(mode)
        // 重建外观子菜单以更新勾选
        statusItem.menu = buildMenu()
    }

    @objc private func openSettings() {
        delegate?.showSettings()
    }
}

private extension NSMenuItem {
    func withTarget(_ target: AnyObject) -> NSMenuItem {
        self.target = target
        return self
    }
}

        // 宽度固定 560；高度根据工具数动态（每项 44pt + 固定部件约 110pt），最大 520

```swift
private struct DragAreaHost: NSViewRepresentable {
    func makeNSView(context: Context) -> DragArea { DragArea() }
    func updateNSView(_ nsView: DragArea, context: Context) {}

    final class DragArea: NSView {
        override func mouseDown(with event: NSEvent) {
            window?.performDrag(with: event)
        }
        override func hitTest(_ point: NSPoint) -> NSView? {
            // 只在按钮之外的空白区响应（按钮 hit test 优先）
            return self
        }
    }
}
```

**注**：DragAreaHost 作为 `background(...)` 插在最底层，SwiftUI 的 Button / IconButton 会先吃掉它们自身 hit 区域的 mouseDown；剩下的空白（标题 / Chip / Spacer 区）会触发 `DragArea.mouseDown` → `window.performDrag`。

// SliceAIKit/Sources/DesignSystem/Components/IconButton.swift
import SwiftUI

/// 通用图标按钮
///
/// 支持两种尺寸：22×22（结果面板 header 等紧凑场景）和 30×30（悬浮工具栏按钮）。
/// 默认 hover 时叠加 `SliceColor.hoverFill`；可指定 active 状态改用 accent 色。
public struct IconButton: View {
    public enum Size {
        case small   // 22×22 / icon 12pt
        case regular // 30×30 / icon 15pt
        var dimension: CGFloat {
            switch self { case .small: return 22; case .regular: return 30 }
        }
        var iconSize: CGFloat {
            switch self { case .small: return 12; case .regular: return 15 }
        }
    }

    let systemName: String?
    let text: String?
    let size: Size
    let isActive: Bool
    let help: String?
    let action: () -> Void

    /// 构造
    /// - Parameters:
    ///   - systemName: SF Symbol 名；与 text 互斥
    ///   - text: 自定义文本 / emoji；与 systemName 互斥
    ///   - size: 尺寸
    ///   - isActive: 激活态（图标染成 accent）
    ///   - help: tooltip
    ///   - action: 点击回调
    public init(systemName: String? = nil, text: String? = nil,
                size: Size = .small, isActive: Bool = false,
                help: String? = nil, action: @escaping () -> Void) {
        self.systemName = systemName
        self.text = text
        self.size = size
        self.isActive = isActive
        self.help = help
        self.action = action
    }

    public var body: some View {
        Button(action: action) {
            Group {
                if let systemName {
                    Image(systemName: systemName)
                        .font(.system(size: size.iconSize, weight: .medium))
                } else if let text {
                    Text(text).font(.system(size: size.iconSize))
                }
            }
            .foregroundColor(isActive ? SliceColor.accent : SliceColor.textSecondary)
            .frame(width: size.dimension, height: size.dimension)
            .hoverHighlight(cornerRadius: SliceRadius.button)
        }
        .pressScale()
        .help(help ?? "")
    }
}

import SwiftUI

/// 胶囊按钮（主 / 次 / 危险三种风格）
public struct PillButton: View {
    public enum Style {
        case primary   // accent 底 + 白字
        case secondary // hoverFill 底 + 主文本
        case danger    // error 底 + 白字
    }

    let title: String
    let icon: String?   // SF Symbol
    let style: Style
    let action: () -> Void

    public init(_ title: String, icon: String? = nil,
                style: Style = .secondary, action: @escaping () -> Void) {
        self.title = title
        self.icon = icon
        self.style = style
        self.action = action
    }

    public var body: some View {
        Button(action: action) {
            HStack(spacing: SliceSpacing.sm) {
                if let icon { Image(systemName: icon).font(.system(size: 11, weight: .medium)) }
                Text(title).font(.system(size: 11.5, weight: .medium))
            }
            .padding(.horizontal, SliceSpacing.lg)
            .padding(.vertical, SliceSpacing.sm)
            .foregroundColor(foreground)
            .background(
                RoundedRectangle(cornerRadius: SliceRadius.button).fill(background)
            )
        }
        .pressScale()
        .buttonStyle(.plain)
    }

    private var foreground: Color {
        switch style {
        case .primary, .danger: return .white
        case .secondary:        return SliceColor.textPrimary
        }
    }
    private var background: Color {
        switch style {
        case .primary:   return SliceColor.accent
        case .secondary: return SliceColor.hoverFill
        case .danger:    return SliceColor.error
        }
    }
}

// SliceAIKit/Sources/DesignSystem/Components/SectionCard.swift
import SwiftUI

/// 圆角白卡分组容器，用于设置窗口内容区
///
/// 用法：
/// ```swift
/// SectionCard("已配置") {
///     SettingsRow("OpenAI") { ... }
///     SettingsRow("DeepSeek") { ... }
/// }
/// ```
public struct SectionCard<Content: View>: View {
    let title: String?
    @ViewBuilder let content: () -> Content

    public init(_ title: String? = nil, @ViewBuilder content: @escaping () -> Content) {
        self.title = title
        self.content = content
    }

    public var body: some View {
        VStack(alignment: .leading, spacing: 0) {
            if let title {
                Text(title.uppercased())
                    .font(SliceFont.captionEmphasis)
                    .kerning(SliceKerning.wide)
                    .foregroundColor(SliceColor.textSecondary)
                    .padding(.bottom, SliceSpacing.base)
            }
            VStack(spacing: 0) { content() }
        }
        .padding(SliceSpacing.xl)
        .background(
            RoundedRectangle(cornerRadius: SliceRadius.card)
                .fill(SliceColor.surface)
                .overlay(
                    RoundedRectangle(cornerRadius: SliceRadius.card)
                        .stroke(SliceColor.border, lineWidth: 0.5)
                )
        )
    }
}
